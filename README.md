# 使用方法(usage):
  輸入任何字元，會得到選單(Enter any character to get the menu)\
  輸入 s + "關鍵字" 取得搜尋結果(Type s + "keyword" to get the result)\
  Ex: s network
# 進階搜尋(6/5更新):
  1.輸入 s + "年代-年代" + "關鍵字"\ 
    可以限制搜尋論文範圍。\
    Ex: s 1999-2018 gene
  2.輸入 s + &"文字" + "關鍵字"\
    & and 文字\
    | or 文字\
    ! not 文字\
    Ex: s &algorithm !finding |dynamic gene
  3. 綜合使用\
    Ex: s 1999-2018 &algorithm !finding |dynamic gene
